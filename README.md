# 删除代码中的注释（REMOVE CODE COMMENT）

## 

## 目前支持的语言有 C、C++、C#、Java、OC、PHP、JS、Lua
* 输入 **-help** 获取帮助
* 输入 **-file**=file full path **-lang**=c# 删除此文件内的注释
* 输入 **-folder**=input folder **-lang**=c# **-ext**=*.cs [**-all**=true] ]删除此文件夹内的所有此语言的文件内的注释, **-all**是可选参数，默认为true，表示包含目录子文件夹
